HOW TO USE
----------
These 8 diagrams are ALREADY built into the site (js/app.js). When you add a
piece in the catalogue manager (admin.html), just pick the matching "Artwork
style" — no files to copy. The piece will show the gold diagram automatically
when it has no photo.

The .svg files in this zip are the same drawings as standalone images, in case
you want to use them elsewhere (print, social, packaging) or hand them to a
designer. To use one as a product photo instead of the line art, drop it into
assets/images/ and set the photo via the admin "Photo" field.
